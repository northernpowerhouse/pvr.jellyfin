# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial implementation of Jellyfin PVR client
- Live TV channel browsing and playback
- Electronic Program Guide (EPG) support
- Recording management (view and delete)
- Timer/scheduled recording management (add and delete)
- Channel groups support
- HLS streaming from Jellyfin
- Automatic addon repository updates via GitHub Actions
- Build system for multiple platforms (Linux, Windows, macOS, Android)

## [1.0.0] - TBD

### Added
- First stable release
- Complete PVR integration with Jellyfin Live TV
- Multi-platform support
- Automated build and deployment pipeline
- Comprehensive documentation
